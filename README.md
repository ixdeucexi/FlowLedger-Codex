# FlowLedger

A polished frontend-only personal finance dashboard built with Next.js, React, and TypeScript.

## Local development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Checks

```bash
npm run typecheck
npm run lint
npm run build
```

## Vercel

Import this repository with the Root Directory set to `./`. Vercel will detect Next.js automatically. The project has no backend services, secrets, or environment variables.
